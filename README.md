# 广电计量股权激励个人所得税计算及分期台账

本项目用于复核和制作广电计量（002967）2023年股票期权与限制性股票激励计划的个人所得税计算及分期缴税台账。

## 项目用途

生成 Excel 工作簿，支持：

- 每名员工每次行权或解禁逐笔录入；
- 同一员工同一纳税年度内多次股权激励所得自动合并计税；
- 按每次事件新增税额的独立 36 个月期限，自定义各次缴税日期和金额；
- 批量管理与单人测算；
- 内置广电计量已公告激励计划参数及来源。

**注意：本项目仅用于公司内部测算和复核，不是自动申报系统。最终申报数据应与主管税务机关备案及申报口径一致。**

## 工作簿结构

工作簿包含 8 个业务 sheet：

1. **使用说明**：录入顺序、颜色规则、计算口径和注意事项。
2. **计划参数**：广电计量激励计划公告参数、员工主数据（员工编号、姓名）。
3. **激励事件明细**：逐笔录入行权/解禁事件，自动计算应纳税所得额、年度累计税额、本次新增税额和 36 个月截止日。
4. **年度计税汇总**：按“员工编号＋纳税年度”汇总年度所得、税额、已缴/未缴金额和最早到期日预警。
5. **分期缴税台账**：登记每笔实际或计划缴税，按事件批次校验额度、截止日和逾期状态。
6. **单人测算**：选择员工后展示事件明细、缴税记录、批次余额和到期/逾期提示。
7. **税率及规则**：综合所得年度税率表、速算扣除数和关键计算规则。
8. **政策及公告**：公司公告和税务政策来源记录。

颜色约定：蓝色为人工输入，灰色为公告参数，绿色为公式结果，红色为异常，黄色为说明。

## 使用流程

1. 在“计划参数”表维护员工编号和姓名（I:J 列）。
2. 在“激励事件明细”表逐笔录入行权或解禁事件。
3. 在“分期缴税台账”表登记计划及实际缴税。
4. 在“年度计税汇总”和“单人测算”表查看汇总结果和余额。
5. 交付前检查校验列、状态列和是否存在公式错误。

## 计算口径

- **股票期权所得**：(行权日收盘价 − 事件日有效行权价) × 本次行权数量。
- **限制性股票所得**：(股票登记日收盘价 + 本批次解禁日收盘价) ÷ 2 × 本批次解禁数量 − 个人实际出资总额 × 本批次解禁数量 ÷ 个人获授总数。
- **年度合并计税**：同一员工同一纳税年度多次股权激励所得合并，单独适用综合所得税率表，速算扣除数只扣一次。
- **本次新增税额**：本次事件后年度累计税额 − 本年度此前事件累计已确认税额。
- **36 个月期限**：每次事件新增税额独立批次，从该事件行权日/解禁日起单独计算 36 个月。
- **离职处理**：在 36 个月期间内离职的，应在离职前缴清全部税款。

关键禁止事项：

- 不得把公司 802 万股计划总数作为个人限制性股票计税分母。
- 不得把授予价当成股票登记日收盘价。
- 不得把同年度累计计税误写成所有事件共用同一个 36 个月截止日。

## 生成及测试命令

依赖：`openpyxl`、`pytest`。

```powershell
# 生成新版工作簿
py src/build_workbook.py

# 或指定输出路径
$env:WORKBOOK_OUTPUT="C:\path\to\output.xlsx"; py src/build_workbook.py

# 运行全部测试（约 80-90 秒，含 Excel COM 验证）
python -m pytest -q

# 运行单个测试文件
pytest tests/test_tax_model.py -v
pytest tests/test_workbook.py -v

# 运行单个测试
pytest tests/test_tax_model.py::test_option_income_for_50000_options -v
```

## 税务政策来源及适用日期

核验基准日：2026 年 6 月 11 日。

主要政策依据：

- 财政部、税务总局公告 2023 年第 25 号：同一纳税年度多次股权激励合并单独计税，政策执行至 2027 年 12 月 31 日。
- 财政部、税务总局公告 2024 年第 2 号：境内上市公司股权激励可在行权/解禁之日起 36 个月内缴税，离职前缴清。
- 国税函〔2009〕461 号：限制性股票所得公式及个人口径分母。
- 财税〔2005〕35 号：股票期权所得公式。

公司公告参数来源：

- 2024-035：激励计划修订说明。
- 2024-043：实际授予公告（授予日 2024-07-01）。
- 2024-045：股票期权授予登记完成公告（登记完成日 2024-07-24）。
- 2024-054：限制性股票授予登记完成公告（上市/登记日 2024-08-27）。
- 2025-002、2025-031、2025-060、2026-036：历次权益分派实施公告及行权价格调整。

完整依据见 `research/公告及政策依据.md` 和 `政策及公告` 工作表。

## 限制与声明

- 不自动抓取未来股价或公告；新增计划需由授权人员更新参数表。
- 不计算股息红利个人所得税。
- 工作簿用于公司内部测算和复核，最终申报数据应与主管税务机关备案及申报口径一致。
- 历史行权价格按当时有效公告处理，不得用最新价格覆盖历史事件。
