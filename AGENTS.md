# 项目说明

本项目用于复核和制作广电计量股权激励个人所得税计算及分期缴税台账。

## 工作原则

- 全部回复、文档和工作表说明使用中文。
- 直接执行并交付结果，不停留在方案层。
- 原始文件只读保留，修改后必须另存新文件。
- 税务结论必须有现行有效政策依据，不凭经验猜测。
- 广电计量激励计划参数必须取自公司正式公告，并记录公告名称、日期和链接。
- 行权日、解禁日、股票登记日的市场价格必须注明可靠来源和日期。
- 政策或公告存在冲突时，优先采用效力更高、发布时间更晚且仍有效的文件，并在复核说明中列明差异。

## 计算口径

- 每名员工每次行权或解禁作为一条事件明细。
- 同一员工同一纳税年度取得两次以上股权激励所得时合并计税。
- 股票期权所得按“行权日市场价减行权价，再乘行权数量”计算。
- 限制性股票所得必须使用股票登记日市场价和本批次解禁日市场价，不得用授予价代替登记日市场价。
- 36个月分期缴税允许自定义日期和金额，不默认按月平均。
- 股息红利个人所得税不纳入本项目的行权、解禁计税模型。
- 不保存员工身份证号，仅使用员工编号和姓名。

## Excel 规则

- 人工输入、公告参数、公式结果和异常提示使用清晰且一致的颜色区分。
- 关键字段设置数据验证，公式单元格和内置参数表设置保护。
- 公式应引用税率表和参数表，避免在多个位置重复硬编码。
- 不保留无用测试工作表、隐藏试算数据或未说明的辅助公式。
- 工作簿交付前必须检查公式错误、重复事件、缺失参数、跨年度归集、超额缴税和逾期状态。
- 使用 Excel 实际打开并重算，检查公式、下拉菜单、筛选、冻结窗格、保护和打印布局。
- 最终工作簿不得存在 `#REF!`、`#DIV/0!`、`#VALUE!`、`#NAME?` 等错误。

## 交付文件

- 新版工作簿默认命名为 `广电计量股权激励个人所得税计算及分期台账-2026版.xlsx`。
- 同时保留复核说明，列明原表错误、修正内容、政策依据和测试结果。
- 所有交付文件放在项目根目录。
